CrossPlaylist 試作

YouTube × ニコニコ × SoundCloud 統合プレイリスト — 公式埋め込み再生（再生数は元動画にカウント）
⚠ file:// で直接開かれています。この状態では YouTube / ニコニコの埋め込み再生が拒否されます（YouTubeエラー153等）。 同じフォルダの start.bat をダブルクリックして起動してください。
ℹ ローカル起動（開発用）です。ニコニコ動画はローカルからの再生を拒否するため、視聴には 公開版 を使ってください。
プレイリストから動画をクリックで再生開始
    クリックで再生 / ドラッグで並び替え / ✕で削除。
    プレイリストはこのブラウザに自動保存（localStorage）。
    一括取り込み: YouTubeはチャンネル/再生リストURL（要APIキー・下記）、
    ニコニコはマイリスト/シリーズ/ユーザーページURLをそのまま貼るだけ（設定不要）。
    ⚙ YouTubeチャンネル/再生リスト取り込み（APIキー設定）

    初回のみ、無料の YouTube Data API キーが必要です（サーバー不要・キーはこのブラウザ内だけに保存されます）。

    1. Google Cloud Console でプロジェクトを作成し「YouTube Data API v3」を有効化
    2. 認証情報 →「APIキーを作成」でキーを取得
    3. 下に貼り付けて保存 → チャンネル/再生リストURLを上の欄に貼るだけで取り込み
    📥 ニコニコの検索結果/ランキング等から取り込む（ブックマークレット）

    マイリスト／シリーズ／ユーザー投稿一覧はURLを上の欄に貼るだけで取り込めます。 検索結果・ランキングなど「動画リンクが並ぶ任意のページ」から取り込みたいときだけ、こちらを使ってください。

    1. 下のリンクをブックマークバーにドラッグして登録
      📥 動画URL収集
    2. ニコニコで取り込みたいページを開く（下までスクロールして全件表示させる）
    3. 登録したブックマークをクリック → 動画URLが全件コピーされる
    4. ここに戻って貼り付け →「＋追加」
